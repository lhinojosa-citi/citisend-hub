Citisend HUB v1.4

Cambios principales:
- Nuevo módulo admin.html para generar y editar proyectos desde una pantalla visual.
- Añadir proyectos nuevos desde Administración y generar data/proyectos.json.
- Edición de dispositivos, instalados, responsable, estado, cliente y Planner.
- Software asociado por proyecto: solo aparece en la ficha el software marcado.
- Galería preparada: las fotos se muestran desde los nombres definidos en proyectos.json.
- Estructura preparada para subir imágenes a proyectos/<id>/galeria/.
- Actualización de versión y accesos a Administración desde el HUB.

Notas:
- GitHub Pages no guarda datos directamente. El admin genera el JSON para copiar en data/proyectos.json.
- Para mostrar fotos, sube los archivos a la carpeta del proyecto y añade su nombre en Administración.
